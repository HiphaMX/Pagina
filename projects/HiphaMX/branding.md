# Brand Identity: HiphaMX

**Organization**: Agency (Main Provider)
**Status**: Internal

## Visual Profile
- **Primary Aesthetic**: Dark Mode, High Tech, Mystic-Energy.
- **Color Palette**:
    - `Primary`: `#06b6d4` (Cyan / Electric)
    - `Secondary`: `#c026d3` (Magenta / Vibrant)
    - `Accent`: `#3b82f6` (Blue / Soft)
    - `Background`: `#050810` (Midnight Black)
- **Typography**: Inter (Sans-serif)

## Key Assets
- **Logos**: Located in `projects/HiphaMX/HiphaIdentidad/Logotipo/`
    - use `HiphaWhite.svg` for dark backgrounds.
    - use `HiphaBlue.svg` for white/light backgrounds.

## Design Patterns
- **Glassmorphism**: 16px blur, 0.5 opacity slate backgrounds.
- **Organic Shapes**: Use of "blobs" and radial gradients to create depth.
- **Hyphae lines**: SVG based biological networks.
